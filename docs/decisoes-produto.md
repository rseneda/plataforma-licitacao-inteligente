# Decisões de Produto

## Decisão 1 — Priorizar geração de TR antes de IA

Motivo:
- Maior impacto imediato
- Base necessária para IA

Impacto:
- Entrega mais rápida de valor
- Redução de complexidade inicial

Motivo:
- Flexibilidade
- Adaptação ao setor público

Impacto:
- Entregas incrementais
