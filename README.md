# Plataforma de Licitação Inteligente

Projeto de portfólio com foco em Product Owner, análise de requisitos e contratações públicas de TI, simulando um sistema real para apoio à elaboração de ETP e Termo de Referência com uso de Inteligência Artificial.

## Problema
Órgãos públicos enfrentam dificuldades na padronização, análise e gestão de contratações de TI, gerando retrabalho, riscos e baixa eficiência.

## Solução Proposta
Sistema que automatiza:
- geração de ETP
- criação de Termo de Referência
- análise de requisitos
- identificação de riscos com IA

## Diferenciais
- Integração entre backlog e contratação pública
- Aplicação prática da Lei 14.133
- Uso de IA para apoio à decisão
- Foco em governança e eficiência

## Competências Demonstradas
- Product Owner (Scrum)
- Levantamento e gestão de requisitos
- Licitações de TI
- Governança de TI
- IA aplicada

## Estrutura
- docs → backlog e visão do produto
- tr → documentos de contratação
- ia → uso de IA no sistema

## Público-alvo
- Órgãos públicos
- Empresas GovTech
- Consultorias de TI
