# Plataforma de Licitação Inteligente

Projeto de portfólio focado na aplicação de práticas de Product Owner, análise de requisitos e contratações públicas de TI.

## Objetivo
Apoiar órgãos públicos na criação, gestão e otimização de contratações de soluções tecnológicas, com uso de inteligência artificial.

## Competências Demonstradas
- Gestão de backlog (Scrum / PO)
- Levantamento e análise de requisitos
- Elaboração de Termo de Referência (Lei 14.133)
- Governança de TI
- Aplicação de IA

## Estrutura do Projeto
- /docs → backlog e visão do produto
- /tr → termo de referência
- /ia → uso de IA
